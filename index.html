Master’s Thesis – AI-Optimized Battery Manufacturing

Fraunhofer FFB Institute, Münster, Germany | Jan 2025 – Oct 2025

As part of my Master’s thesis, I developed a smart scheduling system for battery cell manufacturing by integrating simulation and machine learning. The aim was to optimize production order sequencing based on real-time constraints like urgency, customer dependency, and resource availability, while also predicting expected processing durations for each order.

I built a complete digital workflow combining Siemens Plant Simulation with trained machine learning models through a custom-built Flask API. The models were trained on structured production data to estimate order priority and processing time, using algorithms like CatBoost Classifier and LightGBM Regressor. Predictions were then used to simulate different scheduling strategies and compare outcomes like station utilization and delay reduction.

My work involved preprocessing synthetic and simulation-generated datasets, engineering production-aware features, and evaluating various ML algorithms based on both prediction quality and their ability to generalize to unseen orders. The system was tested end-to-end using real station timings, scrap-rate logic, and scheduling baselines (FIFO, EDD, SPT).

  • Built robust ML pipelines for order prioritization and ETA prediction
  • Integrated trained models into Siemens Plant Simulation via Flask API
  • Compared AI-based scheduling against baseline strategies (FIFO, EDD, ETA)
  • Designed dashboards and tables for live performance monitoring and retraining
  • Worked under real-world constraints like variable scrap rates and processing regimes
